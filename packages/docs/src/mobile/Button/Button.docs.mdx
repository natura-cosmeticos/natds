<Button children="Label" mode="outlined" onPress={() => {}} />

| Name                      | Type                                            | Default                       | Description
| :--                       | :-:                                             | :-:                           | :-:                       |
| children *                | node                                            |                               | Label text of the button.
| mode                      | 'text', 'outlined', 'contained'                 | 'text'                        | Mode of the button.
| color                     | string                                          |                               | Custom text color for flat button, or background color for contained button.
| icon                      | object                                          |                               | Icon to display for the `Button`.
| disabled                  | boolean                                         | false                         | Whether the button is disabled. A disabled button is greyed out and `onPress` is not called on touch.
| contentStyle              | object                                          |                               | Style of button's inner content. Use this prop to apply custom height and width.
| style                     | object                                          |                               | React Native style object.
| labelStyle                | object                                          |                               | Style for the button text.
