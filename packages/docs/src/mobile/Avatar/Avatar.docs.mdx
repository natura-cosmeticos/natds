| Name                      | Type                                            | Default                       | Description
| :--                       | :-:                                             | :-:                           | :-:                       |
| type *                    | 'image', 'text', 'icon'                         |                               | Type of Avatar (image, text or icon).
| label **                  | string                                          |                               | Initials to show as the text in the `Avatar`. Use only in Text Type.
| icon **                   | IconSource                                      |                               | Icon to display for the `Avatar`. Use only in Icon Type.
| size                      | number                                          | 64                            | Size of the avatar.
| style                     | object                                          |                               | React Native style object.
| theme                     | IThemeMobile                                    |                               | Overridable theme provided from the Provider component
| color                     | string                                          |                               | Custom color for the text.
| labelStyle                | object                                          |                               | Style for the title. Use only in Text Type.
